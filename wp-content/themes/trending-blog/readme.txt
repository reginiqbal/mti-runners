== Description ==
Trending Blog is a Clean, Minimal and Responsive WordPress Blog Theme. It follows the trendy design with a combination of elegance. The theme was designed to be used easily of the beginner WordPress users without code knowledge. Trending Blog is fully responsive, meaning it will look great on different devices like tablets, laptops, smartphones, etc. This is the theme you have been waiting for. Live preview at https://demo.themeinwp.com/trending-blog/

WordPress theme "Trending Blog" is a child theme of "galway-lite".
galway-lite Theme is licensed under the GPL3.
